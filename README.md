Chia sẻ với anh em script mình tự làm từ các nguồn nên mạng, dùng để chuyển dữ liệu Zalo bao gồm App, File Recieve, và data sang ổ khác. Mình để mặc định là chép từ C -> D. bác nào muốn để chỗ khác thì edit đường dẫn lại.

Script mở lên là chạy, mọi người giải nén file rồi mở file với quyền admin nhé không chạy file trực tiếp từ trong rar . Sẽ có những lúc "Press any key to continue . . ." mọi người chịu khó nhấn nút bất kì để tiếp tục. Lí do là ở mỗi lệnh chạy mình đều chú thích để mọi người biết script đang chạy cái gì. Chạy xong chương trình sẽ tự khởi động lại Zalo.... thế là xong

LƯU Ý 1 :

1. Vì mặc định là ổ D nên dung lượng ổ D càng lớn càng tốt. Mình từ chuyển cho chị HRM cty mình gần 150 GB.

2. Vì nó di chuyển luôn Zalo Received Files ở thư mục Document nên mọi người nhớ trỏ lại đường dẫn: Cài Đặt -> Tin nhắn -> File được lưu tại thư mục -> Trỏ về thư mục ZALO FILE RECEIVE ở ổ D

3. Đối với bác nào đã thay đổi đường dẫn lưu file trước đó rồi thì trỏ lại đường dẫn lưu file trước đó. Không ảnh hưởng đến script

LƯU Ý 2:
1. Đối với bác nào chỉ di chuyển từ C -> D để giải phóng dung lượng ổ C thì chỉ cần chạy file "BACKUP Zalo.cmd"

2. Đối với bác nào di chuyển từ C -> D để CÀI GUYN hoặc DI CHUYỂN SANG MÁY KHÁC thì PHẢI CÀI ZALO TRƯỚC SAU KHI CÀI GUYN và Chỉnh lại ổ đĩa lưu 3 thư mục thành ổ D RỒI MỚI CHẠY FILE "RESTORE ZALO.cmd". Chạy xong cũng phải trỏ lại đường dẫn Zalo File Recieve nhé. Bác nào cẩn thận thì nên copy lại 1 bản vào USB rồi hãy chạy file RESTORE cho chắc
